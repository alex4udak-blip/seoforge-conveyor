# Asset-стратегия (гибрид генерить/качать) — решение

| Элемент | Генерить/Качать | Источник | Композиция (где) |
|---|---|---|---|
| Hero-баннер | ГЕНЕРИТЬ | Runware (атмосфера казино+гео) | верх страницы, фон под заголовок |
| Фоны/текстуры | ГЕНЕРИТЬ | Runware | секции-разделители |
| Лого платёжек | КАЧАТЬ | simpleicons CDN + Wikimedia (UPI/bKash) | footer payments-row |
| Лого казино | КАЧАТЬ | google favicon API (s2/favicons) из реальных доменов базы | toplist-карточки слева |
| Скрины игр | КАЧАТЬ | провайдер-CDN (Spribe/Pragmatic), casino media | games-секция |
| Звёзды/рейтинг/бейджи | CSS/inline | — | toplist-карточки |
Принцип: точное/узнаваемое (лого, скрины игр) = качать реальное; атмосферное (hero/фон) = генерить.
